A modern browser-based calculator capable of performing basic arithmetic operations with a clean and responsive user interface.

- Decimal calculations
- Division-by-zero handling
- Backspace functionality
- Responsive UI
- Modern glassmorphism design

```
WebDev-L2-ToDoWebApp
```
