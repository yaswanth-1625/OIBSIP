# Alan Turing Tribute Page 🌌

An elegant, highly interactive, and responsive tribute page dedicated to **Alan Turing**, the visionary mathematician, WWII codebreaker, and founding father of computer science and artificial intelligence. Built as part of the **Oasis Infobyte Web Development & Designing Internship** (Level 2, Task 2).

## 🚀 Features

- **SEO Optimized & Semantic Structure**: Designed with semantic HTML5 tags, unique IDs for test compatibility (complies with freeCodeCamp test specifications), and appropriate metadata.
- **Premium Aesthetics**: Features a sleek dark theme with glassmorphic cards, custom typography, and vibrant cyan and violet glowing ambient backdrops.
- **Original Content & Milestones**: Contains 4 distinct biography blocks covering his early life, codebreaking at Bletchley Park, artificial intelligence foundations, and final legacy, plus a chronological milestones timeline.
- **Distinct Quote Block**: Features a styled quote section showcasing Turing's famous words.
- **Fully Responsive Layout**: Built with CSS Grid and Flexbox, with tailored mobile queries to adjust layouts, column structures, and typography size for mobile, tablet, and desktop screens.
- **Custom Artwork**: Includes a stylized cybernetic digital portrait of Turing in the assets folder.

## 🛠️ Tech Stack

- **HTML5** (Semantic Structure)
- **CSS3** (Custom Properties, Flexbox, Grid, Glassmorphism, CSS Transitions & Keyframe Animations)
- **Google Fonts** ('Playfair Display' for Serif accents & headings, 'Outfit' for Sans-Serif body copy)
