# 🔐 Login Authentication System

A modern client-side authentication system developed using HTML, CSS, and JavaScript. The application allows users to register, securely log in, access a protected dashboard, and log out using browser storage for authentication.

Built as part of the **Oasis Infobyte Web Development & Designing Internship (Level 2, Task 4).**

---

## 🚀 Features

- User Registration
- Secure Login
- Password Validation
- Duplicate User Detection
- Protected Dashboard
- Logout Functionality
- Local Storage Authentication
- Responsive User Interface
- Form Validation
- Password Hashing Support

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript
- Local Storage API

## 💻 How to View Locally

1. Clone or download this folder.
2. Open the project directory.
3. Launch **index.html** in any modern browser.
4. Register a new account and log in.
