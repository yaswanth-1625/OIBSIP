# ✅ FocusFlow - To-Do Web App

An elegant, interactive, and responsive task management application built using HTML, CSS, and JavaScript. FocusFlow helps users organize daily activities with features like task creation, editing, completion tracking, deletion, and local storage persistence.

Built as part of the **Oasis Infobyte Web Development & Designing Internship (Level 2, Task 3).**

---

## 🚀 Features

- Add new tasks instantly
- Edit existing tasks
- Mark tasks as completed
- Delete unwanted tasks
- Separate Pending and Completed task sections
- Live task counters
- Data persistence using Local Storage
- Responsive design for desktop, tablet, and mobile devices
- Clean glassmorphism-inspired interface

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Responsive Design, Animations)
- JavaScript (DOM Manipulation, Local Storage)
